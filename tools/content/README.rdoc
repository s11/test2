NAME
  maptool

SYNOPSIS
  maptool import mapfile name=name version=version [moodle_course_category_id=moodle_course_category_id] [options]+

DESCRIPTION
  Will import all data in the given content map CSV file and associate it with the given map and version

PARAMETERS
  mapfile
      The filename of the content map CSV file 
  name=name
  version=version
  moodle_course_category_id=moodle_course_category_id
  --force 
      Force overwriting the specified maps if it already exists 
  --custom_menu=custom_menu
  --help, -h 



Example usage:
