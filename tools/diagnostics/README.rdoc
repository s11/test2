SYNOPSIS

  import --system system_name --scenario scenario_name [--force --practice] csv_filename

DESCRIPTION

  Will import a single scenario into a new or existing diagnostics system.

PARAMETERS

  system_name
    The name of the diagnostics system that the scenario exists in or will be created within

  scenario_name
    The name of the scenario to create or update the content for

  -f, --force
    Force overwriting the specified scenario if it already exists

  -p, --practice
    Flag the scenario as being a practice scenario

  -h, --help
